# Odin Calculator

A simple calculator built with **HTML**, **CSS**, and **JavaScript** as part of **The Odin Project Foundations** curriculum.

## Features

* Basic arithmetic operations (+, −, ×, ÷)
* Chained calculations
* Decimal input support
* Delete (backspace) functionality
* Clear calculator
* Keyboard support
* Previous and current operand display
* Division by zero handling
* Rounded decimal results to prevent display overflow

## Technologies Used

* HTML5
* CSS3 (Flexbox)
* JavaScript (ES6)
